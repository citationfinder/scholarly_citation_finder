### MAG

Download and extract compressed file
```bash
$ cd downloads/harvester/mag
$ curl -O https://academicgraph.blob.core.windows.net/graph-2015-11-06/MicrosoftAcademicGraph.zip
$ 7z x MicrosoftAcademicGraph.zip
```
